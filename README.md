## 系统介绍
开源溯源防伪系统、一物一码系统，适用于各种溯源场景的大、中、小、微规模企业的商用级溯源防伪系统。可追踪记录产品的生命周期各个环节，把产品的环节信息保存在系统中，品牌保护、产品营销、防止假冒伪劣产品、防窜货、产品追踪溯源。应用的场景领域：快销品行业、种植行业、养殖行业、畜牧行业、加工行业、仓储行业、零售行业等。
